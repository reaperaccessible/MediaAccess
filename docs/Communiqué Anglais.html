MediaAccess 1.30 — Official Release

A lightweight, fully accessible audio and video player, designed from day one for blind and visually impaired users.

Announcement

ReaperAccessible is pleased to announce the official release of MediaAccess 1.30, a Windows audio and video player designed from day one to be fully usable by blind and visually impaired users with screen readers. After numerous public iterations and continuous integration of user feedback, MediaAccess today reaches its official stable milestone, ready for daily use.

One player for all your media. MediaAccess brings together in a single lightweight application: local audio file playback, video playback, internet radio listening, podcast subscriptions, direct YouTube search and playback, audio output recording, and a full real-time professional effects chain. No add-ons, no external players, no browser required — everything is integrated and keyboard accessible.

MediaAccess is completely free, open source (GPL-3.0 license), and available immediately as a standalone installer for Windows 11 64-bit.

What is MediaAccess?

MediaAccess is a complete audio and video player for Windows, powered by the professional BASS audio library for audio and libmpv for video. It natively plays the most common audio formats (MP3, FLAC, OGG, M4A, WAV, Opus, AAC, WMA, AIFF, APE, WavPack, ALAC, DSD, MIDI, tracker formats) and all common video formats (MKV, MP4, AVI, MOV, WebM, FLV, TS, M2TS, VOB, 3GP, MPEG, etc.) with full subtitle and multi-language audio track support.

What distinguishes MediaAccess is not a feature list — many audio players offer comparable feature sets. What distinguishes it is that every feature has been designed so that a person navigating with NVDA or JAWS can use it with complete mastery, keyboard only. No compromise: accessibility is not a layer added on top of a mouse-first design. It is the design itself.

Key features

Professional audio playback

  • Native playback of over 25 audio formats through the BASS library and its plugins (FLAC, Opus, AAC, WMA, APE, WavPack, ALAC, DSD, MIDI with SoundFont, tracker formats, audio CD files).
  • M3U / M3U8 / PLS playlist playback with relative path handling.
  • Speech announcement of ID3, Vorbis Comment, MP4 and other tags: title, artist, album, year, genre, track number, comment, bitrate, duration — each tag has its dedicated shortcut (1 through 0).
  • Automatic chapter detection in Ogg, FLAC, Opus, MP3 (ID3v2 CHAP), MP4 files and Podcast 2.0 streams (RSS with <podcast:chapters> tags).
  • Playback up to 400% volume via software amplification option, with configurable step (1 to 25%).

Integrated video playback

  • libmpv video engine bundled — no external installation required.
  • Audio track cycling (useful for multilingual MKV files), subtitle handling (cycle, manual load, disable), aspect ratio cycling (4/3, 16/9, etc.).
  • Fullscreen mode (F11), screenshot capture (Ctrl+Shift+S).
  • Automatic detection of external subtitle files placed next to the video (.srt, .ass, .ssa, .sub, .vtt).

Real-time professional effects chain

MediaAccess offers a complete effects chain, all driveable by keyboard and screen reader.

  • Tempo — slow down or speed up without changing pitch, from -75% to +200%. Three algorithms to choose from: SoundTouch (fast), Google Speedy (optimized for speech), or Signalsmith Stretch (high quality).
  • Pitch — transpose from -12 to +12 semitones without changing speed, ideal for musicians and language learners.
  • Rate — sped-up or slowed-down playback like a vinyl (0.25× to 4×).
  • Reverb — three algorithms (Freeverb, DX8 DirectX, I3DL2 environmental) plus convolution reverb with your own WAV impulse response file.
  • Three-band equalizer with preamp and configurable center frequencies (bass, mid, treble).
  • Full compressor (threshold, ratio, attack, release, output gain).
  • Echo with adjustable delay, feedback, and mix.
  • Stereo width — from mono (0%) to ultra-wide (200%).
  • Center cancel — isolate or remove vocals for karaoke and practice use.
  • 3D binaural or 5.1 audio powered by Steam Audio, with listener position adjustable in X/Y/Z and continuous rotation.
  • Each effect has its own toggle shortcut (Ctrl+1 through Ctrl+=) and all values are saved between sessions.
  • Effect presets — save your favorite combinations and recall them instantly via P.

Two-slider keyboard concept

Control organization revolves entirely around two persistent «virtual sliders».

  • Effects slider: [ and ] cycle through active parameters; up/down arrows adjust the value. The list grows automatically as you enable new effects.
  • Movement slider: , and . change the unit (1 second, 5 seconds, 1 minute, 1 track, 1 chapter, etc.); left/right arrows seek.

This principle separates what to modify from by how much, giving immediate and precise speech feedback with no ambiguity.

Native YouTube integration

  • Integrated YouTube search (Ctrl+Y) — no manual install required.
  • Infinite scroll of results: new results load automatically when you reach the end of the list.
  • MediaAccess signature hybrid playback: playback starts instantly via libmpv while the full file downloads in the background, then automatically swaps to BASS once the download is complete — you get the best of both worlds: instant playback and access to all BASS effects (tempo, pitch, EQ, etc.).
  • Permanent cache (configurable) for instant replays of already-heard tracks — no network cost, even offline.
  • Three cache management options (clear on close, manual clear, size limit with oldest-files-first automatic removal).
  • Embedded chapter detection: even on YouTube, you can navigate by chapters.
  • Optional video mode to watch YouTube directly.
  • Optional YouTube Data API key to speed up search.

Internet radio

  • Three selectable search sources: RadioBrowser (global community database with country filter), TuneIn, and iHeartRadio.
  • Persistent favorites with reordering (Ctrl+arrows), renaming (F2), and M3U/PLS import/export.
  • Live ICY metadata capture (current song title), announced via speech and saved to song history.
  • Quick add of current stream to favorites (Ctrl+D).

Podcasts

  • Complete subscriptions manager with standard OPML import/export.
  • Search the iTunes Podcasts directory.
  • Basic HTTP authentication supported (password-protected RSS feeds).
  • Multi-select episode list (Shift+arrows) with batch downloads.
  • Podcast 2.0 chapters: if the feed contains a <podcast:chapters> tag, MediaAccess fetches them and makes them navigable.
  • Detailed diagnostic when a feed fails to load (URL, HTTP status, error code, received content, tags found).
  • Automatic download organization by feed title.

Recording and scheduling

  • Audio output recording in one shortcut (R) to WAV, MP3, OGG Vorbis or FLAC.
  • Flexible filename template with strftime codes or bilingual friendly tokens ({year}, {année}, {month}, {hour}, etc.).
  • Post-effects capture at full amplitude: listening volume does not affect the recorded file.
  • Event scheduler (Ctrl+S): scheduled playback or recording of a file or radio at a specific date, with daily, weekly, monthly, or weekday/weekend repetition.

Daily productivity

  • Persistent bookmarks (M to add, Ctrl+M to manage) with filter by current file or all bookmarks.
  • Song history (Ctrl+Shift+H): the last 100 titles captured from radio and podcast streams, with date, time, and clipboard copy.
  • 10 recent files in the File menu.
  • Minimize to system tray (Ctrl+H) with full context menu.
  • Configurable global hotkeys: over 35 actions assignable to keyboard combinations active even when MediaAccess does not have focus.
  • Paste from clipboard (Ctrl+V): Ctrl+C a file in Windows Explorer, then Ctrl+V in MediaAccess replaces the playlist and starts playback immediately.

Accessibility — first class, not an afterthought

Screen reader support

  • Compatible with NVDA, JAWS, Window-Eyes, Supernova, and SAPI (fallback) through the bundled Universal Speech library — no configuration needed.
  • All speech announcements are configurable: track changes, volume adjustment, effect adjustment, stream metadata, effect toggles.
  • Every dialog is fully keyboard navigable with Alt+letter accelerators indicated by underlines.

Native bilingual — English and French

  • Automatic Windows language detection on first launch.
  • Manual switch in Options → Playback → Language.
  • Every interface string is translated: menus, dialogs, error messages, NVDA speech announcements, control labels, the user manual itself — nothing is left in English on a French install (and vice versa).

Accessible HTML user manual

  • Press F1 from anywhere to open the complete user manual in your default browser, automatically in the active language.
  • Document compliant with WCAG 2.1 AA standards: proper heading hierarchy, high contrast, «skip to main content» link, navigable table of contents, captioned tables, <kbd> element for each key, automatic dark mode.
  • 22 sections covering 100% of features, with a complete keyboard shortcut reference.

Full autonomy — one installer, everything included

MediaAccess is fully self-contained. The installer ships everything required: audio engine and all its plugins (FLAC, Opus, AAC, MIDI, WMA, etc.), the video engine, YouTube support, the 3D audio library, and screen reader client DLLs (NVDA, JAWS). No third-party component needs to be installed or configured manually — MediaAccess will never ask the user to download or install anything.

Automatic updates

MediaAccess checks for a new version on startup (option can be disabled). If an update is available, an accessible dialog offers to download and install it in one click — no browser or manual intervention needed. The version comparison logic is protected against accidental downgrades.

System requirements

  • Windows 11 64-bit (older versions of Windows are no longer supported by Microsoft and are not targeted).
  • NVDA, JAWS or Windows Narrator screen reader (optional, but strongly recommended for the target audience).
  • Standard 64-bit processor — no recent instruction requirement (Sandy Bridge or newer).
  • ~60 MB disk space for the installer (includes the audio and video engines and all required components).
  • Internet connection for online features (YouTube, radio, podcasts) — not required for local file playback.
