1. Introduction

MediaAccess is a complete, accessible audio and video player for Windows. It is designed for screen-reader users (NVDA, JAWS, Windows Narrator) while remaining pleasant for sighted users. It combines a small footprint with a deep professional effects chain (EQ, reverb, echo, tempo, pitch, stereo width, 3D audio, convolution reverb, and more), all adjustable in real time.

This help is available at any time via F1 or via Help → Manual.

About keyboard shortcuts and your layout

MediaAccess uses the physical key positions rather than the characters those keys produce. The shortcuts in this manual are labeled with QWERTY characters (for example Z, X, C, V, B for the playback transport row).

On a French or Belgian AZERTY keyboard, use the same physical positions (so W, X, C, V, N for the transport row). Same for German QWERTZ keyboards (Y, X, C, V, B), Bépo, Dvorak, Cyrillic layouts, etc.

No configuration is needed: MediaAccess detects the physical key, not the letter. Modifier-based shortcuts (Ctrl+O, Ctrl+S, etc.) remain letter-based mnemonics, like in every Windows application.

2. Getting started

Installation

Download MediaAccessInstaller_version.exe from the GitHub releases page and run it. The installer is bilingual (English/French), auto-detects the Windows display language, and bundles everything required. No separate installs are needed.

First launch

On first launch, MediaAccess:

  • Detects the Windows language and uses French if Windows is French, English otherwise (overridable in Options → Playback → Language);
  • Picks your default audio output device.

Opening a file

Several ways:

  • Ctrl+O or File → Open to pick a file;
  • Ctrl+Shift+O or File → Add Folder to recursively load every audio file in a folder;
  • Ctrl+U to open an http:// or https:// URL;
  • Drag and drop a file on the window, or use "Open with" from Explorer if MediaAccess is registered for the file type (checkbox in Options → Playback).

3. The two virtual sliders

MediaAccess organizes every adjustable parameter around two keyboard "sliders" that always work the same way. Each slider has a key pair to cycle through available items, and a key pair to adjust the value of the chosen item.

Effects slider

Effects slider keys
KeyAction
[ / ]Previous / Next parameter
↓ / ↑Decrease / Increase value
BackspaceReset to default
Ctrl+HomeMinimum value
Ctrl+EndMaximum value

The cycle list is built dynamically from currently-enabled effects. With only Volume enabled, ] does nothing more. Enable the equalizer (Ctrl+7) and ] now walks through Volume → Preamp → Bass → Mid → Treble.

Movement slider

Movement slider keys
KeyAction
, / .Previous / Next seek unit
← / →Seek backward / forward by one unit

Available units: 1 s, 5 s, 10 s, 30 s, 1 min, 5 min, 10 min, 30 min, 1 h, 1 track, 5 tracks, 10 tracks, 1 chapter (if available), 1 subtitle (video only). Enable the ones you want in Options → Movement. By default "5 seconds" and "1 subtitle" are enabled.

The "1 subtitle" unit only appears in the cycle while a video is playing. When selected, ← / → jump to the previous and next subtitle event of the active track. For audio playback (BASS) or DAISY books the cycle skips this unit silently — it has no meaning there. If "Speak subtitles" is enabled, the subtitle you land on is spoken immediately.

Tip: the two-slider concept separates what to modify (the parameter) from by how much (the value). The adjust keys (arrows) give immediate speech feedback through the screen reader.

Jump near the end

Shift+End jumps to 30 seconds before the end of the current track. Useful to preview an ending, skip a bonus track tucked at the end of an album, or catch the wrap-up of an audiobook chapter. The shortcut is a no-op on live streams whose duration is unknown.

Chapters

MediaAccess detects chapters in Ogg/FLAC/Opus files (VorbisComment), MP3 (ID3v2 CHAP), and Podcast 2.0 streams (RSS with <podcast:chapters>). To use them, enable "1 chapter (if available)" in Options → Movement, then select the "chapter" unit with , / . and navigate with ← / →. The current chapter is spoken as "Chapter N: name".

Volume

Volume goes from 0 to 100%. If you enable Options → Playback → Allow volume above 100%, the range extends to 400%. Ctrl+↑ / Ctrl+↓ always adjust volume regardless of the current parameter. The step is configurable (1%, 2%, 5%, 10%, 15%, 20%, 25%).

Technical note: by default, volume is applied after the effects chain, meaning your recordings (R key) capture audio at full amplitude regardless of listening volume. To change this, enable "Legacy volume" in Options → Advanced.

External subtitles

MediaAccess auto-detects subtitle files next to the video file (mpv option sub-auto=fuzzy). To load one manually: Video → Load Subtitle File. Tracks marked "forced" (common on .mkv files where only foreign-language dialogue is subtitled) are supported.

Speak subtitles aloud

MediaAccess can route each subtitle line directly to the screen reader as it appears on screen. This is useful for following a foreign-language film without having to read visually, or simply when you prefer to hear text rather than read it.

To enable: Tools → Options → Speech tab → "Speak subtitles" (off by default). Once the box is checked, play a video whose subtitle track is active and every line will be spoken. Announcement mode is interrupt: if a new line arrives before the previous one finishes, it takes over — necessary behaviour since subtitles flow quickly and queued playback would lag behind the picture.

An assignable action, "Toggle speak subtitles" (Main category, with a Global mirror), lets you turn the feature on or off from a hotkey of your choice. No default shortcut is bound; assign one from Tools → Actions. Each toggle announces "Subtitle speech on" or "off".

Tempo changes speed without changing pitch; Pitch changes pitch without changing speed; Rate changes both (sped-up/slowed-down like a vinyl). Tempo and Rate are disabled for live streams (internet radio).

When you speed up or slow down playback (Tempo or Rate effect), the status bar and the Ctrl+Shift+E, Ctrl+Shift+R, and Ctrl+Shift+T announcements reflect the real wall-clock listening time rather than the raw source duration. 7. Effect presets

You can save your effect combinations and recall them instantly.

  • P opens the presets menu.
  • Choose "Save current preset" to name and save the current effects state.
  • Choose an existing preset to apply it immediately.
  • Choose "Delete" to remove a preset.

A preset captures: the four stream effects (Pitch, Tempo, Rate — not Volume), the reverb algorithm, all enabled DSP effects, and every parameter value. Volume is intentionally not saved to avoid presets changing your listening level.

10. YouTube

MediaAccess has full built-in YouTube support — search, playback, and download — with no external setup required.

Search and playback

  1. Ctrl+Y or File → YouTube opens the dialog.
  2. Type your search and press Enter.
  3. Navigate the results with arrow keys. When you reach the last item, more results load automatically.
  4. Enter plays the selected result.
  5. The Download button saves the video permanently to Downloads\MediaAccess\YouTube\ as m4a audio.

Download with options (choose a format)

Next to the simple Download button, the Download with options... button lets you pick exactly which format to download. It asks YouTube for every available stream and shows them in a list — Format ID, extension, resolution, video codec, audio codec, size and notes — sorted best-first. Use the arrow keys to read each format column by column, then press Enter (or the Download button) to fetch that stream. When you choose a video-only format, MediaAccess automatically merges in the audio with the bundled ffmpeg. The number of available formats is announced when the list opens. The simple Download button still works exactly as before.

Direct YouTube URL

You can paste a YouTube URL (video or playlist) in the search field instead of keywords.

Audio / video mode

By default MediaAccess plays YouTube in audio-only mode (so all BASS effects like tempo and pitch are available). For video, enable video mode in the YouTube options.

Cache

Played videos are cached in %LOCALAPPDATA%\MediaAccess\YouTubeCache\. The cache is permanent — it survives reboots. To clear it: Help → Clear YouTube cache.

YouTube Data API key (optional)

If you enter a YouTube Data API key in Options → YouTube, search uses the official API (faster). Without a key, MediaAccess uses an internal fallback (slower, but no account required). Get a key: console.cloud.google.com.

Test YouTube playback

Help → Test YouTube playback opens a diagnostic dialog confirming that the YouTube extractor and the video engine are functional, plus the location of the log file. Useful when troubleshooting.

13. Recording

R starts/stops recording the audio output.

Formats

  • WAV (lossless)
  • MP3 (configurable bitrate: 128 / 160 / 192 / 224 / 256 / 320 kbps)
  • OGG Vorbis (configurable bitrate)
  • FLAC (lossless)

Filename template

Configure the template in Options → Recording. Supports standard strftime codes (%Y, %m, %d, %H, %M, %S) and friendly tokens:

  • English: {year}, {month}, {day}, {hour}, {min}, {second} / {sec}
  • French: {année} / {annee}, {mois}, {jour}, {heure}, {minute}, {seconde}

Default template: {year}-{month}-{day}_{hour}-{min}-{second}.

Important: by default, recording captures post-effects sound (all enabled effects are included) at full amplitude — the listening volume does not affect the recorded file. If you enable "Legacy volume" in Options → Advanced, volume affects the recording too.

Recording system audio

MediaAccess can record two different things, selected by the Recording source in Options → Recording (or with the Cycle recording source action, to which you can assign a shortcut in the Actions dialog):

  • MediaAccess output — records what MediaAccess itself is playing (the default; everything above applies).
  • Windows system output — records everything Windows is playing through your speakers/headphones (any app: a web player, a game, a video call, another media player), captured via WASAPI loopback and saved with the same format and bitrate settings.

When the source is set to "Windows system output", your usual record key starts and stops the system capture instead. The System device setting is normally left on Automatic (follow MediaAccess output), which captures whichever output device MediaAccess is currently using; you can also force a specific device. If the chosen device is unplugged or disabled while recording, the capture stops cleanly, finalizes the file, and announces that the device was lost.

DRM limitation: protected services such as Netflix, Apple Music and some protected streams may record as silence, because their audio is protected by Windows and cannot be captured through loopback. Unprotected audio records normally.

14. MIDI

Files .mid, .midi, .rmi, and .kar play via BASSMIDI.

SoundFont

MediaAccess ships with FluidR3_GM by Frank Wen (MIT license), the open-source General MIDI gold standard, bundled in the installer. When no custom SoundFont is configured, the bundled SoundFont is used automatically, so MIDI files sound pro out of the box with no setup. The bundled file is installed at <install dir>\lib\FluidR3_GM.sf2.

To use a different SoundFont, open Options → MIDI and browse to a .sf2 or .sf3 file. Clear the field to go back to the bundled FluidR3_GM. If both the field is empty and the bundled file is missing (rare — only happens for portable builds that skipped the SoundFont download), BASSMIDI falls back to its built-in basic synth.

Polyphony

"Maximum voices" controls polyphony (1 to 1000, default 128). Above 512 only matters for very dense MIDI files.

Sinc interpolation

Enables higher-quality interpolation at the cost of more CPU. Recommended for quality listening.

17. Actions and keyboard shortcuts

Every keyboard shortcut in MediaAccess can be remapped through a REAPER-style Actions window, opened with F4 or via Tools → Actions.

Browsing and filtering

The Actions window lists every action in the application, grouped into sections via a category combo:

  • Main: actions of the main window (playback, effects, navigation, etc.);
  • Radio: actions inside the radio browser;
  • YouTube: actions inside the YouTube dialog;
  • Global: system-wide shortcuts that work even when MediaAccess doesn't have focus (see below).

A live Filter field (substring match on action name or assigned shortcut) lets you locate an action quickly. The Find shortcut... button to the right of the filter opens a small dialog: press the combination you're curious about, then Tab to Search — focus jumps to the matching action in the current section.

Assign, edit, or remove a shortcut

Each action shows the list of shortcuts currently bound to it. Three buttons: Add, Edit, Delete. Multiple shortcuts per action are supported (REAPER-style).

If the combination you try to assign is already used by another action, a prompt offers three choices: Replace (removes the prior binding), Cancel, or Assign anyway (both actions then share the same shortcut; the first one encountered in code wins).

Regional and custom keymaps

The installer ships three ready-to-use keymaps: USA, FR-CA (Canadian French), and FR-FR (France French). On first launch MediaAccess auto-detects your Windows keyboard layout and picks the matching keymap.

At the top of the Actions window, a Keymap combo lists every available keymap (the union of the shipped ones and any you have created). The selected item is always the active keymap; changing the selection switches at runtime — the global hotkeys re-register immediately and the choice is persisted. Next to the combo, Import... copies an external .MediaAccessKeyMap file into your user folder, and Delete removes the user copy of the selected keymap (the shipped file is never touched). At the bottom of the window, New keymap... creates a fresh keymap by cloning the active one, and Reset to defaults restores the regional default for your keyboard layout. Files are plain-text .MediaAccessKeyMap files stored under %APPDATA%\MediaAccess\KeyMaps\, easy to share.

Shortcut hints shown in menus auto-update to match the active keymap.

Global section

The Global section is reserved for system-wide shortcuts that fire even when MediaAccess doesn't have focus (implemented via Windows RegisterHotKey). At least one modifier (Ctrl, Alt, Shift, or Win) is required for these shortcuts.

Media keys

Dedicated keyboard media keys (Play/Pause, Stop, Previous, Next) are always globally active, regardless of your custom shortcuts and of the loaded keymap.

18. Command line

MediaAccess accepts a set of switches on its command line, in the style of foobar2000's Foorun tool. When you launch MediaAccess.exe with one or more switches:

  • if a MediaAccess instance is already running, the switch is relayed instantly to that instance (via WM_COPYDATA) and the second process exits silently without opening a window;
  • otherwise, MediaAccess starts normally and then applies the switches. /quit and /hide are exceptions: with no running instance they are simply ignored.

Several switches can be combined in the same call, mixed with file or folder arguments. The order is preserved: load first, play next, adjust the volume last.

This interface is meant for automation: AutoHotkey scripts, Stream Deck buttons, the Windows Task Scheduler, batch files, IR remotes via third-party utilities, or custom screen-reader scripts.

Available switches

Examples

Launch MediaAccess and immediately start playing a folder:

MediaAccess.exe "C:\Music" /play

Load a playlist, start playback, and set the volume to 60%:

MediaAccess.exe "C:\Music\album.m3u" /play /volume:60

Set the volume without showing the window:

MediaAccess.exe /volume:80 /hide

Seek 30 seconds forward in the running instance:

MediaAccess.exe /seek:+30

19. Audio slots

Audio slots let you bookmark up to ten favorite output devices and switch between them with a single keystroke, without opening Options. Ideal for instant switching between your headphones, your speakers, your capture card, your USB interface, or a virtual loopback device.

Each slot remembers a device by name, not by BASS index. You can unplug and replug your USB devices in a different order: the bindings remain valid as long as the name is recognized.

Configuring the slots

Tools → Audio slots… opens the configuration dialog. It contains ten combos, one per slot, listing every BASS-enabled output device detected on the machine; the first entry in each combo is (None), which disables the slot.

If a previously-saved device is currently disconnected when you open the dialog, it appears as "(disconnected) device name" so the binding is preserved and becomes active again automatically as soon as the device is reconnected.

The Refresh device list button re-enumerates the available audio outputs without closing the dialog — handy when you have just plugged in a new device. OK saves your choices, Cancel discards them.

Assigning a shortcut to each slot

Then open the Actions window (F4 or Tools → Actions) and switch to the Global category. There you will find:

  • Audio device slot 1 through Audio device slot 10 — switches the audio engine directly to the device in the indicated slot;
  • Cycle audio output — steps through the configured slots on each press;
  • Speak current audio output — speaks the name of the device currently in use.

Assign each of these actions to the key combination of your choice. Because Global-category shortcuts are handled by RegisterHotKey, they require at least one modifier (Ctrl, Alt, Shift, or Win) and stay active even when MediaAccess does not have focus.

Speech announcements

Each slot action produces an immediate announcement:

  • Successful switch: "Switched to device name";
  • Empty slot: "Slot N is not configured";
  • Disconnected device: "Device not found: name";
  • Speak current output: "Audio output: device name".

Playback position is preserved across the switch: if a track was playing, it continues immediately on the new output with no audible jump.

Use case: live streaming

The original motivation behind audio slots is live streaming. You prepare a file on your default output (heard through your headphones), navigate to the right spot, pause, then press the hotkey bound to the slot pointing at the WDM loopback device of your streaming mixer (for example VoiceMeeter, or OBS's virtual audio output). Pressing Play once more sends the audio straight into the stream, without ever having to touch the system mixer or change the Windows default device.

Keyboard help mode lets you explore shortcuts without executing the action. Ideal for discovering the app or memorizing shortcuts without risking an unintended action.

  • Press F12: MediaAccess announces "Keyboard help on".
  • Press any key, for example Z: "Z: Previous track". The action is NOT executed.
  • Press Ctrl+O: "Ctrl+O: Open file".
  • If the key has no assigned action: "K: no action assigned".
  • Press F12 again to return to normal mode: "Keyboard help off".

Keyboard help mode only operates in the main window. In dialogs (Options, YouTube search, etc.), keys retain their normal behavior so you can type text.

22. Updates

Automatic check

On startup, MediaAccess checks in the background for a new version on GitHub (option Options → Playback → Check for updates on startup, on by default).

Manual check

Help → Check for Updates.

Behavior

If a new version is available, MediaAccess shows a dialog with the current version, the new version, and a Yes/No button. If you accept, the new installer is downloaded then executed silently; MediaAccess closes and reopens in the new version.

MediaAccess will never automatically downgrade: if the remote version is older, no dialog is shown.

23. Troubleshooting

Log file

MediaAccess writes a log to:

%LOCALAPPDATA%\MediaAccess\mediaaccess.log

Rotation is automatic (the old file becomes .log.old at 1 MB).

Loaded plugins

Help → Loaded Plugins shows the list of detected BASS plugins and any that failed to load. Useful if an audio format doesn't work.

Test YouTube playback

Help → Test YouTube playback verifies that the YouTube extractor and video engine are functional.

Audit dialog layout

Help → Audit dialog layout verifies that no text is truncated in the Options dialog for the active language. A quality tool primarily useful for development.

Clear YouTube cache

Help → Clear YouTube cache deletes all files in the YouTube cache. Current size is shown before confirmation.

Set as default media player

Help → Set as default media player opens the Windows default-apps Settings page deep-linked to MediaAccess. From there, click each file type you want MediaAccess to handle (.mp3, .mp4, .mkv, .flac, .mid, etc.) and choose MediaAccess from the list. On Windows 11, a Set default button at the top of the MediaAccess page assigns every supported type at once.

This indirect workflow is a Windows limitation, not a MediaAccess one: Microsoft removed the programmatic "set as default" API in Windows 8 for security reasons. Every media player on the system, including the ones bundled with Windows, hits the same wall — the final choice always belongs to the user, made through Settings.

Reset preferences

To go back to defaults:

  1. Quit MediaAccess.
  2. Delete %APPDATA%\MediaAccess\MediaAccess.ini (installed mode) or MediaAccess.ini next to the executable (portable mode).
  3. Relaunch. Language is re-detected from Windows.

Video load error

If MediaAccess shows "The video playback engine could not be loaded", reinstall MediaAccess. The message includes a technical detail in parentheses useful for diagnosis.

24. DAISY book playback

MediaAccess includes a built-in reader for DAISY accessible audiobooks, comparable to dedicated apps such as Victor Reader or VoiceDream. Books can be opened individually or organized in a library, and all the usual MediaAccess transport, tempo, pitch, EQ, and announcement shortcuts apply during reading.

Supported formats

  • DAISY 2.02 — the most widely distributed format, used by most public libraries and accessible-book services. The book is a folder containing an ncc.html table of contents, SMIL files, and audio files (typically MP3).
  • DAISY 3 (ANSI/NISO Z39.86) — newer specification based on an .opf package and an NCX navigation file.
  • EPUB — every EPUB file is readable, regardless of internal compression. Text-only EPUBs are read aloud through Windows text-to-speech (see below). EPUBs that embed recorded audio via Media Overlays play as full audio books (see below).

Opening a book

  • File → Open book (Ctrl+B): pick an .opf file, an .epub file, or any file inside a DAISY folder (typically ncc.html). MediaAccess identifies the book and starts playback from the beginning, or from the last-read position if the book has been opened before.
  • Tools → Book library (Ctrl+Shift+B): the library lists every registered book with its title, author, and last-read position. Enter reopens the selected book and resumes where you left off.

Configuring the library

Open Tools → Options (Ctrl+,) and go to the Books tab. The tab lets you:

  • Add folder — point MediaAccess to a folder that contains your books. Each subfolder identified as a valid DAISY book is added to the library automatically.
  • Remove folder — stop watching a folder. Books in that folder disappear from the library but are not deleted from disk.
  • Rescan now — re-walk all configured folders, useful after copying new books.

Playback controls

Every transport key used for regular audio also works during a book:

  • Space, X, C, V — play, pause, stop;
  • U — mute;
  • Ctrl+↑ / Ctrl+↓ — volume;
  • ← / → — seek by the active seek unit (1 s, 5 s, 30 s, etc., chosen from the movement slider);
  • Home — beginning of the book;
  • Shift+End — 30 seconds before the end of the book.

Tempo, pitch, and rate effects all work during a book. The elapsed, remaining, and total time announcements reflect the real listening time at the chosen speed (a 10-hour book played at 1.5× shows about 6 h 40 min total).

Structural navigation

A DAISY book exposes a tree of headings (typically chapters, sections, sub-sections), pages, and individual phrases. MediaAccess lets you choose which level to step through:

  • Shift+↑ / Shift+↓ — cycle through navigation levels: Heading 1, Heading 2, ..., Heading 6, Page, Phrase. Each announcement includes the number of items at that level, so you can deduce its meaning (24 entries at level 2 most likely means chapters).
  • Shift+→ / Shift+← — jump to the next or previous item at the active level.
  • G — jump to a specific page. The field accepts a number ("47"), a label ("p. 47", "page 47"), or a roman numeral ("iii"). If the exact page doesn't exist, MediaAccess jumps to the closest one and announces it ("Nearest page: 48").
  • Shift+L — announce the current location ("Chapter 3, page 47").
  • Shift+B — add a bookmark at the current position, optionally with a note.
  • M — open the bookmarks manager for the current book (see below).

Structural navigation now works for EPUBs as well. EPUB 3 files typically include a Navigation Document that exposes the heading hierarchy: MediaAccess reads it directly to drive Shift+↑ / Shift+↓ and Shift+→ / Shift+←. EPUBs that don't ship a Navigation Document fall back to the <h1>–<h6> tags found in the text. Either way, you step through chapters, sections, and sub-sections exactly as on a DAISY book, instead of being limited to paragraph-by-paragraph movement.

Reading progress announcement

Press Shift+P while a book is open to hear your progress: percentage read, current chapter, current page, and estimated remaining time. A typical announcement sounds like: "47 percent, Chapter 3, page 47, about 2 hours 15 minutes remaining."

The remaining-time estimate takes your effective listening speed into account. For audio books, it is computed from the duration of the remaining fragments and the active tempo / rate. For text-only books read by speech synthesis, it is computed from the number of remaining paragraphs and the chosen SAPI rate; slowing down or speeding up the voice updates the estimate immediately.

Skip content during reading

DAISY and EPUB books often mark certain elements as "skippable" because they can interrupt the reading flow: page numbers, producer notes, sidebars, footnotes, and so on. The Books tab of Tools → Options now offers six checkboxes to filter these out during continuous playback:

  • Skip page numbers;
  • Skip notes;
  • Skip sidebars;
  • Skip producer notes (prodnotes);
  • Skip footnotes;
  • Skip references.

MediaAccess recognizes the standard DAISY 2.02 classes (page-normal, optional-prodnote, etc.), the DAISY 3 SMIL classes carried by parent <seq> elements, and EPUB 3 types (epub:type values such as pagebreak, footnote, sidebar, noteref, etc.). When a fragment matches a ticked category, it is automatically stepped over during continuous reading; manual navigation (Shift+→ / Shift+←, go to page) is still free to reach it.

To toggle the filter quickly without opening Options, press Shift+S while a book is playing: this temporarily enables or disables content skipping for the current session. The temporary override is reset the next time you open a book, and the Options checkboxes take over again.

Position memory

MediaAccess saves your position in every book you open. Reopening a book from the library — or simply via Ctrl+B — resumes exactly where you left off, including the active navigation level.

Speech announcements

Ctrl+Shift+E (elapsed), Ctrl+Shift+R (remaining), and Ctrl+Shift+T (total) work as usual during a book and follow tempo/rate adjustments.

Volume difference with music files: DAISY books are typically mastered at a moderate level (around -23 LUFS, the broadcast standard) and will sound noticeably quieter than commercial music MP3s. If you want a louder playback for a book, enable Allow volume above 100% in Options → Playback and raise the volume past 100%.

Text-to-speech for text-only books

Text-only DAISY books and EPUB books without a recorded audio track are read aloud by Windows SAPI 5. The voice is chosen in Tools → Options (Ctrl+,), tab Books, field Text-to-speech voice. The first entry is (Windows default voice); below it are every SAPI voice installed on the machine. To add more voices, open Windows Settings → Time & language → Speech → Voices → Add.

The Rate effect (Ctrl+4, then ↑ / ↓) drives the SAPI speaking rate when a text-only book is being read, exactly as it drives the audio engine when reading a recorded book — one mental model, two engines.

When reading a text-only book via TTS, Shift+→ and Shift+← jump to the next and previous paragraph.

Synchronized text window

For any book that exposes text — recorded audio books with synchronized text and text-only books alike — MediaAccess opens a modeless window that displays the current paragraph as you listen. It opens automatically when the book is loaded; press Ctrl+T at any time to show or hide it.

Three themes are available in Options → Books → Text window theme:

  • Standard — black on white, 12 pt;
  • High contrast — yellow on black, 16 pt;
  • Large — black on white, 24 pt.

If you never want the window to appear, tick Always hide text window in the same Books tab.

Searching inside the book

Press F3 to open the search dialog. Type a search term and press Enter to jump to the first match; press Enter or F3 again to jump to the next match. Escape closes the dialog. Search works on both audio+text books (jumps to the matching clip) and text-only books (jumps to the matching paragraph).

Bookmarks manager

Press M while a book is open to bring up the bookmarks manager. The list shows every bookmark in the current book, labeled by location ("[Chapter 3, page 47] — Key passage"). Three actions are available:

  • Open (Enter or double-click) — jump to the bookmarked position and resume playback.
  • Edit note — edit the text attached to the bookmark.
  • Delete — remove the bookmark (confirmation required).

Bookmarks are sorted in reading order. They live in MediaAccess's SQLite database and persist across sessions, independently of the book's last-read position.

EPUB with Media Overlays

EPUB files that embed synchronized recorded audio (the EPUB 3 "Media Overlays" standard, equivalent to DAISY 3) are played like regular audio books. When such a book is opened, MediaAccess extracts the embedded audio files into a temporary folder and plays them through BASS; the folder is cleaned up automatically when the book is closed. EPUB files without recorded audio continue to be read by the text-to-speech engine described above.

Remapping the shortcuts

Every DAISY shortcut listed above is rebindable via Tools → Actions (F4), section Books.

25. Sleep timer

The sleep timer automatically stops playback after a duration you choose — useful for falling asleep to an audiobook, a podcast, a radio stream, or a movie without having to get back up to turn the sound off. It works with any active playback engine: BASS (audio), libmpv (video), the DAISY/EPUB book player, and the SAPI text-to-speech engine for text-only books.

Starting the timer

The Tools → Sleep timer menu opens a submenu with preset durations: 15, 30, 45, 60, 90, and 120 minutes, plus a Custom duration… entry that opens an input box for any value in minutes. Three shortcuts are also bound by default:

  • F8 — start the timer (cycles through the common presets on repeated presses);
  • Shift+F8 — announce the remaining time;
  • Ctrl+F8 — cancel a running timer.

While the timer is running, the remaining time can be queried at any moment with Shift+F8. When it fires, MediaAccess stops whichever engine is currently playing — whether you are listening to music, watching a video, reading a book, or tuned to a radio.

Optional fade-out

For a softer transition, you can enable a fade-out over the last 30 seconds before the stop. The option lives in Tools → Options (Ctrl+,), tab Playback, checkbox Fade out at end of sleep timer (last 30 seconds). Without the fade-out, the stop is instant.

Remapping the shortcuts

All three shortcuts F8, Shift+F8 and Ctrl+F8 are rebindable via Tools → Actions (F4), section Tools.
