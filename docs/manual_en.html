1. Introduction

MediaAccess is a complete, accessible audio and video player for Windows. It is designed for screen-reader users (NVDA, JAWS) while remaining pleasant for sighted users. It combines a small footprint with a deep professional effects chain (EQ, reverb, echo, tempo, pitch, stereo width, 3D audio, convolution reverb, and more), all adjustable in real time.

This help is available at any time via F1 or via Help → Manual.

2. Getting started

Installation

Download MediaAccessInstaller_version.exe from the GitHub releases page and run it. The installer is bilingual (English/French), auto-detects the Windows display language, and bundles everything required. No separate installs are needed.

First launch

On first launch, MediaAccess:

  • Detects the Windows language and uses French if Windows is French, English otherwise (overridable in Options → Playback → Language);
  • Picks your default audio output device.

Opening a file

Several ways:

  • Ctrl+O or File → Open to pick a file;
  • Ctrl+Shift+O or File → Add Folder to recursively load every audio file in a folder;
  • Ctrl+U to open an http:// or https:// URL;
  • Drag and drop a file on the window, or use "Open with" from Explorer if MediaAccess is registered for the file type (checkbox in Options → Playback).

3. The two virtual sliders

MediaAccess organizes every adjustable parameter around two keyboard "sliders" that always work the same way. Each slider has a key pair to cycle through available items, and a key pair to adjust the value of the chosen item.

Effects slider

Effects slider keys
KeyAction
[ / ]Previous / Next parameter
↓ / ↑Decrease / Increase value
BackspaceReset to default
Ctrl+HomeMinimum value
Ctrl+EndMaximum value

The cycle list is built dynamically from currently-enabled effects. With only Volume enabled, ] does nothing more. Enable the equalizer (Ctrl+7) and ] now walks through Volume → Preamp → Bass → Mid → Treble.

Movement slider

Movement slider keys
KeyAction
, / .Previous / Next seek unit
← / →Seek backward / forward by one unit

Available units: 1 s, 5 s, 10 s, 30 s, 1 min, 5 min, 10 min, 30 min, 1 h, 1 track, 5 tracks, 10 tracks, 1 chapter (if available). Enable the ones you want in Options → Movement. By default only "5 seconds" is enabled.

Tip: the two-slider concept separates what to modify (the parameter) from by how much (the value). The adjust keys (arrows) give immediate speech feedback through the screen reader.

Chapters

MediaAccess detects chapters in Ogg/FLAC/Opus files (VorbisComment), MP3 (ID3v2 CHAP), MP4, and Podcast 2.0 streams (RSS with <podcast:chapters>). To use them, enable "1 chapter (if available)" in Options → Movement, then select the "chapter" unit with , / . and navigate with ← / →. The current chapter is spoken as "Chapter N: name".

Volume

Volume goes from 0 to 100%. If you enable Options → Playback → Allow volume above 100%, the range extends to 400%. Ctrl+↑ / Ctrl+↓ always adjust volume regardless of the current parameter. The step is configurable (1%, 2%, 5%, 10%, 15%, 20%, 25%).

Technical note: by default, volume is applied after the effects chain, meaning your recordings (R key) capture audio at full amplitude regardless of listening volume. To change this, enable "Legacy volume" in Options → Advanced.

6. Effects and DSP

Stream effects

Effects applied to the main stream
EffectShortcutRangeStep
VolumeCtrl+10 to 100% (400% if enabled)2% (configurable)

Tempo changes speed without changing pitch; Pitch changes pitch without changing speed; Rate changes both (sped-up/slowed-down like a vinyl). Tempo and Rate are disabled for live streams (internet radio).

DSP effects

Each DSP effect has a toggle shortcut and adds its parameters to the effects slider when active.

DSP effects
EffectShortcutDescription
ReverbCtrl+5Cycle: Off → Freeverb (musical) → DX8 (DirectX) → I3DL2 (environmental) → Off

Detailed parameters

Freeverb

  • Mix 0–100% (default 30) — dry/wet ratio
  • Room size 0–100% (default 50) — tail length
  • Damping 0–100% (default 50) — high-frequency attenuation in the tail

DX8 Reverb

  • Time 1–3000 ms (default 1000)
  • HF Ratio 0.001–0.999 (default 0.5)
  • Mix -96 to 0 dB (default -10)

I3DL2 Reverb

  • Room Level -10000 to 0 millibels (default -1000)
  • Decay Time 0.1 to 20 s (default 1.49)
  • Diffusion 0–100% (default 100)
  • Density 0–100% (default 100)

Echo

  • Delay 10–2000 ms (default 300)
  • Feedback 0–90% (default 40)
  • Mix 0–100% (default 30)

EQ

  • Preamp -15 to 0 dB
  • Bass -15 to +15 dB (default center: 50 Hz)
  • Mid -15 to +15 dB (default center: 1000 Hz)
  • Treble -15 to +15 dB (default center: 12000 Hz)

Center frequencies are configurable in Options → Advanced.

Compressor

  • Threshold -60 to 0 dB (default -20)
  • Ratio 1:1 to 20:1 (default 4:1)
  • Attack 0.01 to 500 ms (default 20)
  • Release 10 to 2000 ms (default 200)
  • Output gain -20 to +20 dB

3D Audio (Steam Audio)

  • Blend 0–100% (default 100)
  • Width 15–90° (default 45)
  • Rotation -180 to +180° (default 0, wraps around)
  • Mode Binaural / 5.1 Surround
  • Rear speaker On/Off (5.1 mode only)
  • X/Y/Z position -50 to +50 (default 0,0,0)

7. Effect presets

You can save your effect combinations and recall them instantly.

  • P opens the presets menu.
  • Choose "Save current preset" to name and save the current effects state.
  • Choose an existing preset to apply it immediately.
  • Choose "Delete" to remove a preset.

A preset captures: the four stream effects (Pitch, Tempo, Rate — not Volume), the reverb algorithm, all enabled DSP effects, and every parameter value. Volume is intentionally not saved to avoid presets changing your listening level.

9. Bookmarks and history

Bookmarks

Bookmark management
KeyAction
MAdd a bookmark at the current position
Ctrl+MOpen the bookmarks manager

In the manager, a filter shows only the current file's bookmarks or all bookmarks. Enter jumps to the bookmark, Delete removes it. Bookmarks are persisted in a SQLite database.

Recent files

The File → Recent Files submenu lists up to 10 recently opened files.

Song history

Ctrl+Shift+H or File → Song History opens the list of the last 100 song titles captured from internet streams (radio, streamed podcasts). Date, time, and title. Ctrl+C copies the selected title.

The history contains only streams — local file playback is not recorded. Consecutive duplicates are auto-removed.

10. YouTube

MediaAccess has full built-in YouTube support — search, playback, and download — with no external setup required.

Search and playback

  1. Ctrl+Y or File → YouTube opens the dialog.
  2. Type your search and press Enter.
  3. Navigate the results with arrow keys. When you reach the last item, more results load automatically.
  4. Enter plays the selected result.
  5. The Download button saves the video permanently to Downloads\MediaAccess\YouTube\.

Direct YouTube URL

You can paste a YouTube URL (video or playlist) in the search field instead of keywords.

Audio / video mode

By default MediaAccess plays YouTube in audio-only mode (so all BASS effects like tempo and pitch are available). For video, enable video mode in the YouTube options.

Cache

Played videos are cached in %LOCALAPPDATA%\MediaAccess\YouTubeCache\. The cache is permanent — it survives reboots. To clear it: Help → Clear YouTube cache.

YouTube Data API key (optional)

If you enter a YouTube Data API key in Options → YouTube, search uses the official API (faster). Without a key, MediaAccess uses an internal fallback (slower, but no account required). Get a key: console.cloud.google.com.

Test YouTube playback

Help → Test YouTube playback opens a diagnostic dialog confirming that the YouTube extractor and the video engine are functional, plus the location of the log file. Useful when troubleshooting.

11. Internet radio

Ctrl+R or File → Radio opens the radio browser.

Two tabs

  • Favorites: your saved stations.
  • Search: live search across three selectable sources:
    • RadioBrowser: community database with country filter.
    • TuneIn: search the TuneIn catalog.
    • iHeartRadio: search iHeartRadio.

Country filter (RadioBrowser)

With RadioBrowser selected, a country combo appears. Select a country to narrow results, or leave "(Any)". The list is sorted by station count (the combo is editable; you can type directly). You can search with a country selected and an empty search field — you'll get every station from that country sorted by popularity.

12. Podcasts

Ctrl+Shift+P or File → Podcasts opens the manager.

Two tabs

  • Subscriptions: three panels — subscriptions list on the left, episodes at top right, description at bottom right.
  • Search: search the iTunes Podcast directory.

Manage subscriptions

  • Add by URL: a dialog asks for the RSS feed URL, optionally username/password (basic HTTP auth supported).
  • Subscribe (from the Search tab): adds the selected iTunes podcast.
  • Import / Export OPML: standard format to exchange subscriptions between podcast clients.
  • Delete on a subscription: confirmation then unsubscribe.
  • Ctrl+↑ / Ctrl+↓: reorder subscriptions.
  • Refresh button: reloads episodes from the selected feed.

Episode list (multi-select)

Episode navigation
KeyAction
Shift+↑/↓Extend selection
Ctrl+↑/↓Move focus without changing selection
Ctrl+SpaceToggle selection of focused item
EnterPlay all selected episodes (or just one)
Double-clickPlay only the clicked episode

Downloads

  • Download button: downloads all selected episodes to the queue.
  • Download All button: downloads every episode in the current feed.
  • Destination: Options → Downloads → Folder. With "Organize by feed", a subfolder is created per podcast.
  • Already-present files are skipped; duplicate names in a single batch get a "(2)", "(3)", etc. suffix.

Podcast 2.0 chapters

If the feed contains a <podcast:chapters url="..."/> tag, MediaAccess fetches the chapter JSON before playback and makes them navigable via the "chapter" unit in the movement slider.

Failed feed diagnostic

When loading a feed fails, a detailed dialog appears: URL, HTTP status, Win32/WinInet error code, bytes received, number of <item> tags found, number of episodes extracted, and a response body preview. Useful for diagnosing a malformed feed.

13. Recording

R starts/stops recording the audio output.

Formats

  • WAV (lossless)
  • MP3 (configurable bitrate: 128 / 160 / 192 / 224 / 256 / 320 kbps)
  • OGG Vorbis (configurable bitrate)
  • FLAC (lossless)

Filename template

Configure the template in Options → Recording. Supports standard strftime codes (%Y, %m, %d, %H, %M, %S) and friendly tokens:

  • English: {year}, {month}, {day}, {hour}, {min}, {second} / {sec}
  • French: {année} / {annee}, {mois}, {jour}, {heure}, {minute}, {seconde}

Default template: {year}-{month}-{day}_{hour}-{min}-{second}.

Important: by default, recording captures post-effects sound (all enabled effects are included) at full amplitude — the listening volume does not affect the recorded file. If you enable "Legacy volume" in Options → Advanced, volume affects the recording too.

14. MIDI

Files .mid, .midi, .rmi, and .kar play via BASSMIDI.

SoundFont

In Options → MIDI, specify the path to a SoundFont file (.sf2 or .sf3). Without a SoundFont, BASSMIDI uses its default synthesizer.

Polyphony

"Maximum voices" controls polyphony (1 to 1000, default 128). Above 512 only matters for very dense MIDI files.

Sinc interpolation

Enables higher-quality interpolation at the cost of more CPU. Recommended for quality listening.

17. Global hotkeys

Global hotkeys are active even when MediaAccess doesn't have focus.

Configuration

  1. Open Options → Global Hotkeys.
  2. Check "Enable global hotkeys".
  3. Click Add and select an action plus a key combination (at least one Ctrl, Alt, or Shift modifier is required).
  4. Edit or Remove act on the selection.

Media keys

Dedicated media keys (Play/Pause, Stop, Previous, Next) are always globally active, regardless of your custom hotkeys.

Available actions

Over 35 actions available: Play, Pause, Stop, Next, Previous, Seek Backward/Forward, Volume Up/Down, Toggle Mute, Speak Elapsed/Remaining/Total/Now Playing, Toggle each effect, Toggle Recording, Show/Hide Window, Open YouTube, Toggle Shuffle, etc.

  • Show track name in window title
  • Auto-advance to next playlist item
  • Follow playback in playlist dialog

  • Allow multiple instances
  • Register all supported file types
  • Rewind on pause (ms)
  • Volume step: 1 / 2 / 5 / 10 / 15 / 20 / 25%
  • Language: English / Français (applies on restart)

Recording tab

Output folder, filename template, format (WAV/MP3/OGG/FLAC), bitrate. See section 13.

Downloads tab

  • Downloads folder (for podcasts)
  • Organize downloads into folders by feed title

Speech tab

  • Announce track changes (off by default)
  • Speak volume when adjusted (on by default)
  • Speak effect value when adjusted (on by default)
  • Announce YouTube effects activation (on by default) — when a YouTube video plays in hybrid mode, playback starts via libmpv (no effects) then swaps to BASS once the download completes. This announcement tells you when the BASS effects (tempo, pitch, EQ, etc.) become available. Uncheck if you find the announcement repetitive.

Movement tab

Checkboxes to enable each movement unit: 1 s, 5 s, 10 s, 30 s, 1 min, 5 min, 10 min, 30 min, 1 h, 1 track, 5 tracks, 10 tracks, 1 chapter. By default, only 5 s is enabled.

Global Hotkeys tab

See section 17.

Effects tab

Checkboxes to add Volume, Pitch, Tempo, Rate to the effects slider, choice of rate step (0.01× or semitone), checkboxes to enable each DSP effect, reverb algorithm selection, and impulse response file path for convolution reverb.

Advanced tab

  • Buffer size: 100 / 200 / 300 / 500 / 1000 / 2000 ms
  • Update period: 5 / 10 / 20 / 50 / 100 / 200 ms
  • Tempo/pitch algorithm: SoundTouch / Speedy / Signalsmith
  • EQ frequencies: Bass (20-500 Hz), Mid (200-5000 Hz), Treble (2000-20000 Hz)
  • Legacy volume (faster but affects recordings)
  • Disable batch delay
  • Reset station/podcast order to alphabetical (button)

YouTube tab

  • YouTube Data API key (optional, speeds up search)
  • Clear cache when MediaAccess closes (off by default) — every YouTube video played is cached permanently in %LOCALAPPDATA%\MediaAccess\YouTubeCache\ for instant replays. Check this if you want everything wiped on each close (useful on shared machines or to save disk space). Note: replays will then be slow (re-download every time).
  • Clear cache now — button to wipe the cache immediately. Asks for confirmation, shows the current size, then reports the number of removed files.
  • Limit cache size (MB, 0 = unlimited) — maximum allowed size. When the cache exceeds this limit, the oldest files are removed at the next startup. Suggested: 500 MB for light use, 2000 MB for daily use.

SoundTouch tab

  • Anti-alias filter
  • AA filter length: 8 / 16 / 32 / 64 / 128 taps
  • Quick algorithm (lower quality, less CPU)
  • Prevent clicks (reduces artifacts)
  • Interpolation: Linear / Cubic / Shannon
  • Sequence (ms): 0-200 (0 = automatic)
  • Seek window (ms): 0-100 (0 = automatic)
  • Overlap (ms): 0-50

Speedy tab

Nonlinear speedup: optimizes acceleration for speech (compresses vowels less than consonants).

Signalsmith tab

  • Quality preset: Default (higher quality) / Cheaper (lower CPU)
  • Tonality limit (Hz, 0 = auto): 0-20000. Suggested: 0 for auto, 4000-8000 for speech, 8000-16000 for music.

MIDI tab

See section 14.

19. Accessibility

Supported screen readers

MediaAccess uses the Universal Speech library, which auto-detects the active screen reader:

  • NVDA via nvdaControllerClient64.dll
  • JAWS via SAAPI64.dll
  • Window-Eyes, Supernova, SAPI (fallback)

No configuration needed — if a screen reader is active, MediaAccess speaks.

What is announced automatically

  • Track change (if enabled)
  • Stream metadata (song title on radio/podcast)
  • Volume change (if enabled)
  • Effect adjustment (if enabled)
  • Movement unit change
  • Chapters
  • Errors (always)
  • Repeat mode, shuffle, mute, recording (on every toggle)

On-demand announcements

  • Ctrl+Shift+E: elapsed time
  • Ctrl+Shift+R: remaining time
  • Ctrl+Shift+T: total duration
  • 1 to 0: ID3 tags (see section 4)

Keyboard navigation

All dialogs are keyboard-navigable. Tab and Shift+Tab cycle between controls. Alt+letter accelerators are indicated by an underlined letter in each label.

Automatic bilingualism

The interface adapts to the Windows language on first launch. You can change it manually in Options → Playback → Language. All dialogs, menus, error messages, and speech announcements are available in English and French.

20. Updates

Automatic check

On startup, MediaAccess checks in the background for a new version on GitHub (option Options → Playback → Check for updates on startup, on by default).

Manual check

Help → Check for Updates.

Behavior

If a new version is available, MediaAccess shows a dialog with the current version, the new version, and a Yes/No button. If you accept, the new installer is downloaded then executed silently; MediaAccess closes and reopens in the new version.

MediaAccess will never automatically downgrade: if the remote version is older, no dialog is shown.

21. Troubleshooting

Log file

MediaAccess writes a log to:

%LOCALAPPDATA%\MediaAccess\mediaaccess.log

Rotation is automatic (the old file becomes .log.old at 1 MB).

Loaded plugins

Help → Loaded Plugins shows the list of detected BASS plugins and any that failed to load. Useful if an audio format doesn't work.

Test YouTube playback

Help → Test YouTube playback verifies that the YouTube extractor and video engine are functional.

Audit dialog layout

Help → Audit dialog layout verifies that no text is truncated in the Options dialog for the active language. A quality tool primarily useful for development.

Clear YouTube cache

Help → Clear YouTube cache deletes all files in the YouTube cache. Current size is shown before confirmation.

Reset preferences

To go back to defaults:

  1. Quit MediaAccess.
  2. Delete %APPDATA%\MediaAccess\MediaAccess.ini (installed mode) or MediaAccess.ini next to the executable (portable mode).
  3. Relaunch. Language is re-detected from Windows.

Video load error

If MediaAccess shows "The video playback engine could not be loaded", reinstall MediaAccess. The message includes a technical detail in parentheses useful for diagnosis.
