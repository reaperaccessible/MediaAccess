1.49
DAISY book reader (Phase 1). MediaAccess now plays DAISY 2.02 and DAISY 3 audio books like a Victor Reader or VoiceDream. Open a book via File > Open book (a folder containing NCC.html, an .opf file, or an .epub) or browse your library via Tools > Book library. The Library is built from folders you register in Préférences (coming) — but you can also rescan from the library window itself. Each book remembers your last position automatically; bookmarks support optional notes (Shift+B prompts for one). Navigation uses Shift+Up/Down to cycle through nav levels (heading 1 through 6, page, phrase) and Shift+Left/Right to jump backward/forward at that level. G opens a Go-to-page prompt. Shift+L announces where you are ("Chapter 3, page 47"). EPUB without audio and text-only DAISY books are recognized and added to the library — full text-to-speech playback for these will arrive in Phase 2.

1.55
DAISY/EPUB book reader Phase 4 — sleep timer, reading-progress announcement, EPUB chapter navigation, and skippable content. New Tools > Sleep timer cascade with presets (15/30/45/60/90/120 minutes) and a custom-duration dialog (F8); Shift+F8 cancels, Ctrl+F8 announces the time remaining. Optional 30-second fadeout during the final stretch keeps the end gentle on the ears (Préférences > Lecture). Press Shift+P inside a book to hear "47 percent, Chapter 3, page 47, about 2 hours 15 minutes remaining". EPUBs that ship a Navigation Document (most EPUB 3 files) now expose proper chapter/section navigation: Shift+Up/Down cycles the heading levels exactly like a DAISY book, Shift+Left/Right jumps between chapters; EPUBs without a Nav Document fall back to h1-h6 headings detected in the text. The new "Skip during continuous reading" preferences (Préférences > Livres) let listeners bypass page numbers, footnotes, sidebars, producer notes, notes or references during continuous playback — Shift+S inside a book toggles the filter off temporarily.

1.53
DAISY/EPUB book reader Phase 3 — bookmark management, smarter page navigation and EPUB Media Overlays. Press M while a book is open to see every bookmark in that book: jump to any entry, edit its note, or delete it. The Go-to-page prompt (G) now accepts plain numbers, "p. 47" / "page 47" / "47", and roman numerals like "iii" — and falls back to the nearest existing page when an exact match doesn't exist (handy for books with sparse page markers). EPUB books that include Media Overlays (synchronized audio + text, the EPUB equivalent of DAISY 3) are now parsed end-to-end: audio entries are extracted from inside the .epub to a per-book temp folder and played through BASS just like a DAISY book. Text-only EPUBs continue to use SAPI TTS as before.

1.52
Fix the auto-update silently failing to overwrite MediaAccess.exe because the running process held a file lock. Users on 1.49 / 1.50 / 1.51 saw the in-app updater download, run the installer, report success, then on the next launch find the same old version still installed and the same update offer pop up again. The installer now ships with CloseApplications=yes + AppMutex=MediaAccessSingleInstance so Inno Setup's Restart Manager closes the running MediaAccess gracefully before file copy. The updater also delays the installer launch by 2 seconds via a cmd wrapper to give the main window time to fully exit, so both ends of the chain are robust regardless of update-checker age. This is the lock-up bug, not a new feature — apologies to everyone who saw the loop.

1.51
Fix the update checker offering an update over and over when MediaAccess was already up to date. The check previously also fired when versions matched but commit SHAs differed (a corner case for shipping same-version hotfix rebuilds), which surfaced false positives whenever an installer was built from a parent commit and uploaded under a release tag pointing at the next commit. The check now compares versions only — bumping the version is how a release ships, no surprises.

1.50
DAISY Phase 2 — text-to-speech, synchronized text window, and full-text search. MediaAccess now plays text-only DAISY books and EPUB books that don't include recorded audio by synthesizing them with Windows SAPI 5. Choose the voice in Preferences > Books. The same Rate effect (Ctrl+4 then Up/Down arrows) speeds up or slows down both BASS audio and the SAPI voice, so the experience is consistent across audio and text-only books. EPUB books are now extracted from their zip archive via miniz, so any EPUB regardless of compression mode is supported. A new synchronized text window (Ctrl+T to show / hide, opens automatically when a book with text is loaded — disable in Preferences > Books with "Always hide text window") displays the current paragraph; three themes are available (Standard, High contrast, Large). F3 opens a search dialog; type, press Enter to jump to the first match, press Enter or F3 again to jump to the next match. Shift+Left / Shift+Right also work for paragraph navigation in text-only books.

1.49
DAISY book reader. MediaAccess now plays accessible audiobooks in DAISY 2.02 and DAISY 3 formats — the same standard used by Victor Reader Stream and VoiceDream. Open a book via File > Open book (Ctrl+B) or browse your collection via Tools > Book library (Ctrl+Shift+B). Configure folders to scan in Preferences > Books — every DAISY book found in those folders is auto-added to the library. Structural navigation: Shift+Up/Down cycles through navigation levels (heading 1-6, page, phrase) with item-count announcement, Shift+Left/Right jumps to the previous/next item at that level, Shift+L announces the current location (chapter, page), G jumps to a specific page, Shift+B adds a bookmark with optional note. Each book remembers its last-read position automatically and resumes there next time. Audio MP3 clips play through the standard BASS engine, so all the usual MediaAccess goodies apply: volume control, tempo, speed-adjusted time displays, F12 keyboard help. Text-only DAISY and EPUB books are recognized and added to the library, but their text-to-speech playback comes in a future update. Tools > Actions includes a new "Books" category for rebinding every shortcut.
Fix French accents (and other non-ASCII characters) showing as literal "î" sequences in YouTube search results. The internal JSON parser only decoded \n and \" — now it handles every JSON escape including \uXXXX with surrogate pair support, so "Chaîne", "café", emojis and CJK characters all display correctly.

REAPER-style Actions / Keymap window. Every keyboard shortcut is now user-customizable. Tools > Actions (or F4) opens a window where you browse all actions by section (Main / Radio / YouTube / Global), filter live, assign multiple shortcuts per action, search by shortcut, and load / save / reset .MediaAccessKeyMap text files. Global hotkeys unified. The keymap's Global category now actually registers via RegisterHotKey() so system-wide shortcuts (active even when MediaAccess does not have focus) fire correctly. Pre-1.41 [Hotkeys] INI entries are automatically migrated into the new keymap on first launch.
The old "Global Hotkeys" tab in Preferences has been removed. Manage all shortcuts (windowed and global) from Tools > Actions.
New Tools menu in the menu bar, containing the Actions item.
Menu items auto-update their displayed keyboard hints to reflect the active keymap.
Conflict prompt when assigning a shortcut already in use: Replace / Cancel / Assign anyway (REAPER-style).
Keymap saves go to %APPDATA% if the install dir is read-only.
NVDA / JAWS / Narrator announce captured shortcuts in real time inside the Add / Edit / Find dialogs.
